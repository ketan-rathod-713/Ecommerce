 - Headless Ui [Website](https://headlessui.com/react/menu)
 - React Hook Form [Documentation](https://www.react-hook-form.com/)


## Installation

Install all npm dependencies and start react.

```bash
  npm install 
  json-server --watch data.json --port 8080
  npm start
```
